# 🔮 Tarot da Andressa Nunes

Bem-vindo ao repositório oficial do site de **Tarot da Andressa Nunes** ✨

Este projeto foi desenvolvido para apresentar os serviços de leitura de tarot, oferecendo uma experiência simples, bonita e acessível para quem busca orientação espiritual e autoconhecimento.

---

## 🌙 Sobre o projeto

O site foi criado com o objetivo de:

- Apresentar os serviços de tarot
- Facilitar o contato com clientes
- Oferecer uma forma rápida de escolher planos de atendimento
- Criar uma experiência visual agradável e mística ✨
